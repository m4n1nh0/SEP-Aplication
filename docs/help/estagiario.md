# Guia de uso - Estagiario

## Objetivo do perfil

O estagiario usa o sistema para cadastrar disponibilidade de atendimento, consultar agenda, acompanhar pacientes vinculados, registrar prontuarios e solicitar alta clinica quando indicado.

## Menu disponivel

- `Inicio`
- `Meus Horarios`
- `Minha Agenda`
- `Meus Pacientes`
- `Solicitar Alta`

## Inicio

Mostra o resumo do dia.

Use para:

- Conferir consultas de hoje.
- Ver proximos horarios.
- Identificar atendimentos pendentes.

Rotina:

1. Abrir `Inicio` no comeco do turno.
2. Conferir consultas.
3. Preparar-se para cada atendimento.
4. Depois do atendimento, registrar prontuario.

## Meus Horarios

Use para informar disponibilidade de atendimento.

Fluxo:

1. Acesse `Meus Horarios`.
2. Escolha o dia da semana.
3. Informe hora inicial.
4. Informe hora final.
5. Envie para aprovacao.

Depois:

- O horario fica `pendente`.
- Coordenador ou supervisor revisa.
- Quando aprovado, o horario pode ser usado para agendamentos.

Cadastre apenas horarios em que realmente podera atender.

## Minha Agenda

Use para acompanhar suas consultas.

Informacoes exibidas:

- Data e hora.
- Paciente.
- Modalidade.
- Sala ou link.
- Status.
- Sessao.

### Status comuns

- `pendente`: consulta criada, aguardando confirmacao.
- `confirmado`: consulta confirmada.
- `realizado`: atendimento registrado por prontuario.
- `cancelado_admin`: consulta cancelada pela equipe.
- `cancelado_paciente`: consulta cancelada pelo paciente.
- `faltou`: paciente nao compareceu, registrado pela equipe.

Antes da consulta:

- Confira paciente, horario e sala.
- Revise historico e prontuarios anteriores quando houver.
- Confirme orientacoes recebidas do supervisor.

Depois da consulta:

- Registre o prontuario.
- Confirme se a sessao foi marcada como `realizado`.

O estagiario nao deve registrar falta pela agenda. Se o paciente nao compareceu, comunique recepcao, coordenacao ou supervisao para que a ausencia seja registrada no fluxo correto.

## Meus Pacientes

Use para acompanhar pacientes vinculados.

E possivel:

- Ver lista de pacientes.
- Consultar dados de contato.
- Ver urgencia.
- Ver total de sessoes.
- Ver ultima sessao.
- Abrir prontuarios quando disponivel.

Um paciente aparece nessa lista quando existe vinculo ativo entre voce e ele.

## Prontuarios

O prontuario registra informacoes clinicas da sessao.

Campos comuns:

- Queixa principal.
- Descricao da sessao.
- Intervencoes.
- Evolucao.
- Plano para proxima sessao.

Fluxo:

1. Acesse a consulta ou paciente vinculado.
2. Abra o formulario de prontuario.
3. Preencha dados da sessao com linguagem tecnica e objetiva.
4. Salve.
5. Confira se a consulta passou para `realizado`.

Boas praticas:

- Registre logo apos a sessao.
- Use linguagem clara e respeitosa.
- Evite julgamentos pessoais.
- Nao inclua informacoes irrelevantes.
- Siga orientacoes do supervisor.

## Solicitar Alta

Use quando houver indicacao de encerramento do acompanhamento.

Antes:

- Converse com o supervisor.
- Revise evolucao do caso.
- Confirme orientacao ao paciente.
- Verifique plano de encaminhamento quando necessario.

Fluxo:

1. Acesse `Solicitar Alta`.
2. Selecione o paciente.
3. Informe justificativa.
4. Descreva evolucao e plano de encerramento.
5. Envie para avaliacao.

Depois:

- A solicitacao fica pendente.
- O supervisor aprova ou rejeita.
- Se aprovada, o vinculo e encerrado e o paciente passa para `alta`.
- Se rejeitada, o acompanhamento continua.

## Situacoes comuns

Horario nao aparece para agendamento:

- Pode estar pendente de aprovacao.
- Pode ter sido rejeitado.
- Pode nao cruzar com disponibilidade de pacientes.

Paciente nao aparece em `Meus Pacientes`:

- Confirme se o vinculo foi criado.
- Verifique com recepcao ou coordenacao se o paciente foi agendado para voce.

Consulta aparece como `pendente`:

- A consulta existe, mas ainda nao foi confirmada.
- Confira com recepcao, paciente ou supervisao se necessario.

Paciente nao compareceu:

- Nao registre prontuario como atendimento realizado.
- Informe a equipe para registrar falta.

Nao consigo registrar prontuario:

- Confirme se a consulta existe.
- Confirme se o paciente esta vinculado a voce.
- Confirme se esta acessando pelo perfil correto.

## Checklist de cada atendimento

Antes:

1. Conferir agenda.
2. Revisar paciente e sessao anterior.
3. Confirmar sala/modalidade.
4. Alinhar duvidas com supervisor.

Depois:

1. Registrar prontuario.
2. Conferir status da sessao.
3. Anotar pontos para supervisao.
4. Atualizar plano para proxima sessao.

## Conduta e privacidade

- Use o sistema apenas para atividades do SEP.
- Nunca compartilhe senha.
- Nao deixe prontuarios abertos em computador compartilhado.
- Saia da conta ao finalizar.
- Trate dados de pacientes com sigilo profissional e academico.
